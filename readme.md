# dotfiles

Minimalist, bar-less Wayland environment for Arch Linux.

## Core Stack
- **Compositors:** Hyprland & Niri
- **Editor:** Neovim
- **Shell:** Zsh + Oh My Posh
- **Terminal:** Kitty + Tmux
- **Keyboard:** Kanata (remaps)
- **File Manager:** Yazi

## Progress
- [x] Core Compositors (Hyprland / Niri)
- [x] Terminal Stack (Kitty, Tmux, Zsh)
- [x] Editor (Neovim)
- [x] Keyboard Mapping (Kanata)
- [x] System Locking (Hyprlock / Swaylock)
- [x] Display Manager (SDDM)
- [x] File Management (Yazi)
- [ ] Notifications
- [ ] OSD (On-Screen Display)

> **Note:** This setup is intentionally bar-less. Information is accessed via keybinds or terminal utilities.

## Installation
```bash
git clone https://github.com/Cards29/dotfiles.git ~/dotfiles
cd ~/dotfiles
stow <package_name>
```
